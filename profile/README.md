# thecontextcache™

**A shared project memory for teams that work with AI.**
Capture the decisions and findings from your AI chats once — recall them as a paste-ready context pack in any AI tool.

---

## 🔎 What we're building

Teams today face a fragmented AI landscape:

- **Tool fragmentation** — different people use different AI tools (ChatGPT, Claude, Cursor, Copilot). Insights found in one don't transfer to another.
- **Scattered context** — project knowledge lives across dozens of chat threads, each siloed, with no shared memory.
- **Context-window limits** — every model forgets earlier decisions on long projects.
- **No curation** — auto-save-everything tools bury the high-signal insights in noise.

ContextCache is a **curated, cross-tool, team-owned** memory layer that fixes all four: capture what matters, recall it anywhere.

## ✨ How it works

1. **Capture** decisions/findings from AI chats (browser extension, MCP, CLI, or web) — with a preview and automatic secret redaction.
2. **Curate** — an extraction step turns raw chat into clean, typed memory cards you approve in an Inbox.
3. **Recall** — a hybrid engine (full-text + vector + recency) returns a formatted memory pack to paste into any AI, or your AI reads it natively via MCP.

## 🚀 Status

🚧 **Invite-only alpha.** Passwordless passkey sign-in, browser extension, MCP server, CLI, and self-hostable Docker stack.
